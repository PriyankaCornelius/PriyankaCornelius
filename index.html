Projects

Property Management

A comprehensive Property Management platform designed to streamline operations for property managers, owners, maintenance personnel, and tenants. As a frontend developer, I crafted the user interface using React.js and Material UI, creating a seamless experience. I implemented role-based navigation to ensure each user persona can efficiently access their relevant data and tasks.

Key Features:

  • Owners can track rent payments, manage properties, and view financial data.
  • Property managers oversee clients, owners, renters, and maintenance requests, with an intuitive graphical view of owner satisfaction.
  • Renters and tenants can explore new properties, manage maintenance requests, pay rent, and access documents effortlessly.
  • Maintenance personnel efficiently handle quote requests, complete tickets, generate invoices, and track past jobs and schedules.

Technologies used: React.js, Material UI, Python, SQL, REST API
